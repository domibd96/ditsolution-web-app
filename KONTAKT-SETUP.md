# Cicero IT – Kontaktformular Setup

Wie bei CraftsbyPam: Node.js-Backend mit Nodemailer.

## Start

```powershell
cd c:\Users\domin\Desktop\UG
npm install
npm start
```

Danach im Browser öffnen: **http://localhost:3001**

Das Kontaktformular sendet E-Mails über die Gmail-Konfiguration in `server.js` (wie bei CraftsbyPam).

## E-Mail-Konfiguration

In `server.js` sind die gleichen Gmail-Daten wie bei CraftsbyPam hinterlegt.  
Empfänger der Formular-Nachrichten: `dominik.trausmuth@gmail.com`.
